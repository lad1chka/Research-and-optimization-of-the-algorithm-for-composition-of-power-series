**Исследование и оптимизация алгоритма композиции степенных рядов.**

Москвитин Владислав. Прикладной анализ данных и искусственный интеллект. 4 курс.

Работа опирается на статью:
> Bostan, Lecerf, Schost, *A Faster Algorithm for Power Series Composition*, arXiv:2404.05177, 2024.


Композиция степенных рядов — это вычисление формального ряда  
**h(x) = f(g(x))**,  
где f(x) и g(x) — формальные степенные ряды.
Задача заключается в вычислении первых `n` коэффициентов композиции с минимальной вычислительной сложностью.
Классические алгоритмы имеют квадратичную сложность `O(n²)`, что делает их непрактичными для больших `n`, поэтому для решения данной задачи нужен более современный подход
## Планы
- Изучить существующие алгоритмы композиции степенных рядов
- Разобрать и формализовать алгоритм из приложенной статьи
- Реализовать базовую и оптимизированную версии алгоритма. Провести сравнение по асимптотике и практическому времени работы
- Провести профилирование, исследовать, какие места в коде наиболее нагружены по времени и памяти. Оптимизировать код
- Исследовать, насколько большую погрешность вычислений имеет алгоритм, оптимизировать погрешность вычислений.
- Продемонстрировать полученные результаты и реализованный алгоритм
